Cara Kerja Browser

Apa itu Browser?

Di zaman serba teknologi seperti ini, semakin banyak orang yang sudah menggunakan internet di dalam kehidupan sehari-harinya, termasuk dalam melakukan sebuah pekerjaan.  Ketika mendengar istilah browser otomatis kamu juga terfikirikan internet, kan? Yup. Tapi, apa itu web browser?

Pengertian browser adalah salah satu jenis perangkat lunak atau software yang pada umumnya di gunakan dan dimanfaatkan untuk membuka suatu halaman di situs internet. Browser juga di sebut dengan software yang berfungsi sebagai penerima, pengakses, dan penyaji berbagai informasi di internet.  Di dalam web browser terdapat konten berupa gambar, audio, video, dan halaman web. Konten-konten tersebut di hubungkan menggunakan hyperlink dan diklasifikasikan dengan URL (Uniform Resources Identifiers). 

Web browser pertama hadir dengan nama WorldWideWeb dan di temukan oleh Tim Berners-Lee pada tahun 1990, yang kemudian di ubah menjadi Nexus. Namun, browser grafis pertama dan yang banyak di gunakan untuk membantu popularitas naik ke Internet adalah NCSA Mosaic. Dengan web browser lah pengguna internet dapat berselancar dan mencari apa saja informasi atau data yang di perlukan. Browser menjadi dasar setiap pencarian informasi di seluruh penjuru dunia sebagai tempat yang menampung serta menampilkan semua yang kamu minta pada kolom pencarian.

Apa Perbedaan Web Browser dan Search Engine?

Meski terkesan hampir sama, namun browser berbeda dengan search engine. Browser memang dapat digunakan untuk mencari informasi di internet, namun browser bukanlah “search engine” itu sendiri.

Surfshark menyatakan bahwa browser memungkinkan pengguna melihat, menemukan, dan mengakses website. Sementara search engine menggunakan algoritma guna menentukan ranking di website itu. Sebagai contoh, ketika seseorang mengetikkan “apa itu digital marketing?” di address bar, bagian yang bertugas mencari jawaban atas pertanyaan itu adalah search engine. Browser bertindak sebagai program atau medianya.

Apa Fungsi Browser?

Setelah sedikit paham mengenai apa itu web browser, yuk kita bahas tentang kegunaan dan fungsi web browser itu sendiri. Fungsi web browser adalah:

1. Mempermudah dalam mencari informasi di internet

Ketika kamu ingin browsing atau berselancar di internet, pasti akan membutuhkan web browser untuk medianya. Benar bukan, Sob? Nah, web browser adalah software yang fungsinya mengarahkan user menuju alamat website yang kamu minta. 

2.Menyimpan data-data di internet

Saat menggunakan browser, pasti kamu pernah mendengar bookmark? Bookmark ini sendiri adalah fitur dari browser yang berfungsi sebagai penyimpanan data-data di internet yang suatu saat bisa kamu buka lagi tanpa perlu mengunjungi search engine.

3. Membuka file dengan ekstensi tertentu

Selanjutnya, fungsi web browser adalah sebagai media untuk membuka file dengan ekstensi tertentu. Misalnya, ketika kamu membuka pdf dengan browser. Atau bagi kamu yang suka otak-atik coding, pasti kamu akan melihat hasilnya melalui web browser. Web browser dapat membaca file ekstensi HTML, CSS, JS, PHP, dan sebagainya.

4. Mendukung pemakai search engine

Seperti yang sudah di bahas tadi, browser dan search engine tidak bisa di pisahkan. Sebab search engine adalah bagian dari web yang bisa memudahkan pengguna untuk mencari data maupun informasi di internet.

5. Menjaga keamanan website

Apakah kamu tahu siapa yang menjalankan proses verifikasi atau authentication? Yup, benar sekali, web browser! Jadi, dapat di katakan bahwa fungsi web browser adalah memastikan halaman website itu aman atau berbahaya untuk di gunakan. Sebab saat ini banyak website sudah terkena malware maupun di pasang software khusus untuk mencuri data pada internet.

Cara Kerja Browser

Web browser adalah software yang bisa membantumu menemukan website yang ingin kamu kunjungi dengan cara berikut:

  1. User sedang mengakses website dengan mengetikkan alamat pada situs maupun URL di address bar. Misal youtube.com. 
  2. Lalu, web browser menerima request tersebut dan akan melakukan fetching atau pengambilan data pada DNS server. 
  3. Data yang sudah di ambil tadi berupa IP dari request yang diketikkan user (www.youtube.com). Web browser telah mendapat IP dari www.youtube.com.
  4. Web browser mengakses server dengan IP yang sudah didapatkan dari DNS server.
  5. Server memberikan data konten dari www.youtube.com dalam bentuk HTML dan file lain seperti PHP dan CSS.
  6. Terakhir, web browser akan menampilkan konten yang sesuai dengan request user. 

Kira-kira seperti itulah cara kerja web browser dalam melaksanakan tugasnya sebagai media untuk mencari informasi di internet.
